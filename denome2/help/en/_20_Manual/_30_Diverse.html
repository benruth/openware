Error-Handling

If problems occur you can try to resynch Denome 2 with the connected device by requesting the current State via the Menu-Entry: Settings->Update

Denome 2 reacts to Events that are transmitted by the Device. Sometimes those Events can contain obsolete informations which can result in unexpected changes of the user-interface.

Maximal Volume

The maximum volume depends on the user specified setting inside the Device and the current channel-settings.

The scale of the volume-dial can change dynamically. Hence changes to specific channel-volumes can affect the display of the volume-dial.



Reset Settings

Changes to the userinterface (position, size, ...) or the connected device-address are automatically saved. The Changes are saved in the following file:
%APPDATA%\denome2\state.sav respectively
C:\Users\%USER%\AppData\Roaming\denome2\state.sav (default)

The removal of this file will reset Denome 2 to Factory-Settings. The connection to the device will be lost; remember to save the Host-Address.
